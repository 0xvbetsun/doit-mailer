### Installing

Cloning project:
```
git clone https://github.com/VBetsun/doit-mailer.git /path/to/project
```
Navigate to folder:
```
cd /path/to/project
```
Install packages:
```
composer install
```
Run built-in server:
```
php artisan serve
```

Enjoy on [this url](http://127.0.0.1:8000) :)
